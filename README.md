# Frontend Projects Collection

A collection of frontend projects built using HTML, CSS, and JavaScript to improve UI/UX design skills and JavaScript fundamentals.

## Projects Included

| Project Name | Description |
|-------------|-------------|
| 3D Boxes Background | Interactive 3D box animation effect |
| Animated Countdown | Countdown animation using JavaScript |
| Auto Text Effect | Automatic typing text effect |
| Blurry Loading | Loading screen with blur transition |
| Content Placeholder | Skeleton loading animation |
| Drink Water Tracker | Daily water intake tracker |
| Form Input Wave | Animated form input labels |
| GitHub Profiles | Search and display GitHub user profiles |
| Hidden Search Widget | Expandable search bar |
| Movie App | Movie search application using API |
| Netflix Mobile Navigation | Netflix-style mobile navigation menu |
| Notes App | Create and manage notes |
| Password Generator | Generate secure random passwords |
| Quiz App | Interactive quiz application |
| Todo List | Task management application |

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
